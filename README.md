## 📐 1. Grid System (গ্রিড সিস্টেম)

<div align="center">

| Class | Description |
|-------|-------------|
| `.container` | কেন্দ্রীভূত, রেসপনসিভ ফিক্সড-উইডথ কন্টেইনার |
| `.container-fluid` | পূর্ণ-প্রস্থ কন্টেইনার |
| `.row` | গ্রিড রো |
| `.col` | সমান-প্রস্থ কলাম (অটো লেআউট) |
| `.col-6` | ১২-গ্রিডের মধ্যে ৬ (সব ডিভাইসে) |
| `.col-md-4` | মিডিয়াম ডিভাইসে ৪ কলাম |
| `.col-lg-3` | লার্জ ডিভাইসে ৩ কলাম |
| `g-0` / `g-3` | গাটার (gap) — 0 থেকে 5 পর্যন্ত |
| `.row-cols-3` | প্রতি রো-তে ৩টি কলাম |

</div>
